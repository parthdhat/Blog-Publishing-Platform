# Multi-User Blog Publishing Platform

A full-stack content publishing system with role-based workflows.

## Features
- Draft → Review → Publish workflow
- Editor moderation dashboard
- Role-based access control
- Server Actions & App Router (Next.js 16)
- PostgreSQL database

## Tech Stack
- Next.js (App Router)
- TypeScript
- PostgreSQL
- Tailwind CSS

## Roles
- Author
- Editor
